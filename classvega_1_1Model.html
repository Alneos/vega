void vega::Model::makeCellsFromDirectMatrices ( )
private

Build the corresponding cells of ElementSets of Direct Matrices (DISCRETE_0D, DISCRETE_1D, STIFFNESS_MATRIX, MASS_MATRIX, DAMPING_MATRIX).

They generally don't have associated cells/cellgroup. This may causes problems in a generic cell writer, such as in SYSTUS.

void vega::Model::splitDirectMatrices ( const unsigned int  sizeMax)
private

Split all direct matrices whose sizes is greater than sizeMax into smaller matrices.

All data is kept, just divided into various elements. Used, for example, by Systus, where cells can't have more than 20 nodes. Must be called before makeCellsFromDirectMatrices().

void vega::Model::splitElementsByDOFS ( )
private

Some elementSet can hold very general elements, acting on various DOFs of the corresponding cells.

For example, Nastran PELAS1 can regroup CELAS1 cells which "spring" in various DOFS (DXtoDX, DYtoDZ, etc) whereas Systus can only have one spring direction by 1902 Part. This method splits the ElementSets to have only one direction in each.
