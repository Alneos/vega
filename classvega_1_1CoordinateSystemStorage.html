Detailed Description

Compute the local spheric base (ur, utheta, uphi) corresponding to point.

Point must be expressed in the reference cartesian coordinate system. Not done Translate a vector, expressed in this coordinate system (ur, utheta, uphi), to its global counterpart. Warning, it does not take the origin into account, so do NOT use this to convert coordinates.This class allows the model to store Coordinate System BEFORE they are created. Each CS, user-defined or model-defined (Orientation) get a unique Position number when it's created or reserved (whatever comes first). This Position number is stored into CellData and NodeData, for further use.

When the CS is created, we associate its model ID (Vega Identifiable number) and user ID (provided by the input file) to the POSITION. When the CS is reserved, we associate its user ID to the POSITION.
