const VectorialValue vega::CylindricalCoordinateSystem::getLocalEulerAnglesIntrinsicZYX ( const CoordinateSystem *  rcs = nullptr) const

Compute the Euler Angles (PSI,THETA,PHI) of the local base, around the axes (OZ, OY, OX) of the coordinate system RCS.

If no rcs is provided, the global coordinate system is used.
